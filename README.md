# vsgx.github.io

This is my personal website showcasing my projects, hobbies, and contact information.  
The site is being developed as I learn front-end technologies step by step.

## Roadmap

### 1. Basic Structure
- Set up the basic structure of the website. 
- Create an `index.html` with basic content. 
- Add a simple navigation bar.

### 2. Layout and Styling
- Learn CSS to style the website.
- Add a layout with a header, main content, and footer.
- Implement a mobile-first design approach.

### 3. JavaScript Integration
- Implement interactive elements using JavaScript.
- Add a theme switcher (light/dark mode).

### 4. Advanced Features
- Work on responsiveness and design improvements.
- Optimize for performance and SEO.
- Add additional pages (about me, portfolio, blog, etc.).
